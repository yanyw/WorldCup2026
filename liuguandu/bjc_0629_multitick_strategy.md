## 二、⚠️ 组合矛盾诊断 (关键新增)

### 问题：6笔头寸 = 同一方向的重复押注

初版推荐了6笔交易（BTTS NO + U2.5 + RSA U0.5 + CAN U0.5 + SELL CAN First + NEITHER First），
但联合场景分析揭示：

```
┌──────────────────────────────────────────────────────────────┐
│                   头寸盈亏相关性矩阵                            │
├──────────────┬────────┬────────┬──────────┬──────────┬────────┤
│              │BTTS NO │ U2.5   │ RSA U0.5 │ CAN U0.5 │NEITHER │
├──────────────┼────────┼────────┼──────────┼──────────┼────────┤
│ BTTS NO      │  1.00  │  0.78  │  0.90⚠️  │  0.59    │  0.48  │
│ U2.5         │  0.78  │  1.00  │  0.69    │  0.57    │  0.48  │
│ RSA U0.5     │  0.90⚠️ │  0.69  │  1.00    │  0.48    │  0.59  │
│ CAN U0.5     │  0.59  │  0.57  │  0.48    │  1.00    │  0.90⚠️│
│ NEITHER First│  0.48  │  0.48  │  0.59    │  0.90⚠️  │  1.00  │
└──────────────┴────────┴────────┴──────────┴──────────┴────────┘
  ⚠️ >0.80 = 高度重叠，边际信息量接近零
```

### 三个致命矛盾

**矛盾 #1：6笔头寸几乎全同向** — 所有交易都在赌"低比分"。当比分达到 1-2/2-1/2-2 及以上时，
6笔头寸**全部同时亏损**。这≈一个 **$900 做空进球** 的集中押注，毫无多样化。

**矛盾 #2：CAN U0.5 是冗余的** — 当 CAN U0.5 赢(加拿大零进球)，BTTS NO 基本上也赢了(因为至少一队零进球)。
两者相关性 0.59，CAN U0.5 的边际信息量几乎为零。花 $150 买了一份已经免费拥有的保险。

**矛盾 #3：NEITHER First 是 RSA U0.5 的严格子集** — NEITHER (0-0) 只有在南非不进球的场景下才能赢。
RSA U0.5 已经覆盖了这个逻辑。花 $50 买了 RSA U0.5 的一个极端子集。

|---------|:----:|:----:|:--------:|
| 0-0 | 11.5% | **+$1,747** | 6/6 ✅ |
| 1-0, 2-0 | 9.7% | +$802 | 4/6 |
| 0-1, 0-2 | 30.8% | +$260 | 3/6 |

> 原组合期望值: **$+117.4** (ROI +13.0%) — 但 **25.6%** 概率血本无归。

## 三、✅ 修正后组合 (4笔, $700)

去掉 CAN U0.5 和 NEITHER First 两个冗余头寸，保留核心三件套 + 一个独立头寸：

|:-:|------|:----:|:---:|:------:|:--------:|:------:|:----:|:------:|

| 2 | **U2.5** | 买入 | $200 | 58¢ | 63.3% | 58.0% | +5.3% | ⭐ |

### 核心三件套 (⭐) 的互补逻辑

三者不会同时亏损。不同比分下的盈亏分化证明了真正的多样化：

| 比分 | 概率 | BTTS NO | U2.5 | RSA U0.5 | 组合盈亏 |
|------|:----:|:-------:|:----:|:--------:|:--------:|
| 0-0 | 11.5% | ✅ +$170 | ✅ +$145 | ✅ +$244 | **+$559** |
| 0-1 | 17.5% | ✅ +$170 | ✅ +$145 | ✅ +$244 | **+$559** |
| 0-2 | 13.3% | ✅ +$170 | ✅ +$145 | ✅ +$244 | **+$559** |
| 1-0 | 7.4% | ✅ +$170 | ✅ +$145 | ❌ -$200 | **+$115** |
| 1-1 | 11.2% | ❌ -$200 | ✅ +$145 | ❌ -$200 | **-$255** |
| 1-2 | 8.5% | ❌ -$200 | ❌ -$200 | ❌ -$200 | **-$600** |

> 0-1/0-2/0-0 三场景(42.3%)全赢；1-1(11.2%)有U2.5对冲；高比分(25.6%)全输。
> 这才是真实的对冲结构——不是"永不亏损"，而是**亏损场景被明确限定了**。

### 🔵 SELL CAN First 的独立价值

与核心三件套的相关性分别仅 0.59/0.57/0.48。当加拿大被对手率先进球（如 RSA 1-0 领先），
BTTS NO + U2.5 可能仍然盈利，但 SELL CAN First 独立盈利。提供了一层额外保护。

## 七、教训总结：为什么初版组合是危险的

| 初版问题 | 影响 | 修正 |
|---------|------|------|
| 6笔头寸高度同向 | 25.6%概率 $900 全亏 | → 4笔 $700 |
| CAN U0.5 冗余 | $150 买了一份已有敞口 | → 去掉 |
| NEITHER 是 RSA U0.5 子集 | $50 重复押注 0-0 | → 去掉 |
| 简单加总 EV(+$138) | 忽略联合概率，严重高估 | → 联合场景 EV $+77.2 |
| "15.3% ROI" 误导 | 假设独立性，实际仅 11.0% | → 如实呈现 |

> **核心教训**: 在一个足球比赛上，所有"低比分"相关盘口本质上是同一个观点。
> 你无法通过开6个不同名字的头寸来对冲——它们只是同一个赌注的6张面孔。
